# Pristinix – Doorstep Detailing & Car Wash

Pristinix is a luxury automotive doorstep detailing and car wash web application built with **Next.js 15 (App Router)**, **Tailwind CSS**, and **Lucide React**.

## 🏎️ Features
- **Signature Services**: Express Wash, Wash + Interior Refresh, Interior Deep Cleaning, Exterior Deep Detailing, Complete Detailing.
- **Paint Protection Coatings**: Teflon, Ceramic (9H), Graphene (10H), and Full Body PPF.
- **Monthly Care Plans**: Gloss (Basic), Shine (Premium), and Pristine (Elite) membership subscriptions.
- **Direct WhatsApp Integration**: Instant one-click booking links for every package pre-configured to `+91 62819 84964`.
- **Responsive Premium Dark Theme**: Custom glassmorphism, red accent lighting, and optimized mobile drawer menu.

## 🚀 Getting Started

First, install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
